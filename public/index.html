Clonewars

A clone of Codewars, a training ground for developers.

The goal of this project was to create a pixel perfect clone of select features of the Codewars site. The front end was built with Angular, and the back end with Node/Express and SQL. One of the great challenges was properly testing user generated code, and funneling this code back and forth between the front and back ends. Another great challenge was to get the design as close as possible to the original in a limited time frame.

See the site here, or check out the GitHub repo. For a thorough overview of the project, watch our presentation on YouTube

The Great Social Accumulator

A single-page app designed to aid influencer marketing

This application was built as a tool for people searching for potential influencer marketers. The user inputs Facebook, Twitter, and Instagram links, and the app parses out user and post ids from the link then collects useful data from the proper API. It has a simple Angular 1 back-end, but where the app really shines is the Node.js server. It uses sockets to lower the back-end load while collecting potentially thousands of data. We made it a priority to keep development as test-driven as possible, so the various api calls are especially robust due to a large Mocha/Chai test suite.

See the site here, or check out the GitHub repo.
